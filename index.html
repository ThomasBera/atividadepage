HTML significa Hypertext Markup Language e é focado em estruturar conteúdo com tags. O HTML é responsável por construir a estrutura básica do site, sendo o “esqueleto” dele, permitindo incluir as estruturas de títulos, parágrafos, imagens e outros elementos de uma página.

CSS é a sigla para Cascading Style Sheets (ou Folha de Estilos em Cascata, em tradução livre). Trata-se de uma linguagem de formatação de conteúdo muito utilizada em conjunto com o HTML na construção de páginas da internet, sendo responsável pelo “visual” do site.
